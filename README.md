Real time Object Detection ด้วย MobileNet SSD ระบบตรวจจับและระบุวัตถุ ในวิดิโอจากกล้องแบบ Real-time
โดยใช้ Deep Learning Model และ OpenCV
โดยจุดประสงค์ของโปรเจกต์นี้คือ พัฒนาระบบที่สามารถตรวจจับและระบุประเภทของวัตถุ ในวิดีโอจากกล้องแบบ Real-time โดยใช้ Pre-trained Model 
ที่ไม่ต้องฝึกใหม่ แสดงผลเป็นกรอบสี่เหลี่ยมพร้อมชื่อและเปอร์เซ็นต์ความมั่นใจ

 ผลลัพธ์ที่ได้
 - ป้ายชื่อแสดงประเภทวัตถุและเปอร์เซ็นต์ความมั่นใจ เช่น PERSON (92.34%)
 - กรอบสี่เหลี่ยมล้อมรอบวัตถุที่ตรวจพบฃ
 - หน้าต่างวิดีโอแบบ Real-time จากกล้อง
